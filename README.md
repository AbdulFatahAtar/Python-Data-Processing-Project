# Python Data Processing Project (IY499)

## Project Overview

This project was developed as part of the **IY499 module** and demonstrates the use of Python for processing and analysing structured data.

The program applies three main data processing techniques:

- Filtering
- Aggregation
- Sorting

These techniques are commonly used in database systems and data analysis workflows.

---

## Project Features

The program performs the following operations:

### Filtering

The system extracts specific records from the dataset based on defined conditions.

Filtering helps reduce large datasets to only the relevant information required for analysis.
